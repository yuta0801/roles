# roles

ユーザーが自分に好きな役割をつけるためのコマンドを追加するDiscordボット

## Description

ボットの権限より低い権限をコマンドにより各ユーザーが自分につけることをできるようにするボット

## Requirement

- discord.js
- dotenv

## Usage

1. clone `git clone https://github.com/yuta0801/roles.git`
1. install `npm install`
1. `.env.sample`を`.env`にコピーし、編集
1. 実行 `node index.js`
